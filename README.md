# PIFCapsule
![image](https://github.com/liguge/PIFCapsule/blob/main/image/Fig1.png)
We have made a summary statement of the above-mentioned articles of the same type, all of which belong to classic cases of the first-layer interpretable paradigm. Regarding the problems that still exist in this paradigm, we propose some new ideas and perspectives from three perspectives: the first-layer interpretable module, the feature extraction module, and the loss function.
